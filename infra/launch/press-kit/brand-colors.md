## Logo Usage

- Minimum size: 120px wide
- Clear space: 0.5× logo height on all sides
- Do not recolor the mark (use provided SVG)
- Do not use on backgrounds with < 3:1 contrast ratio

## Font Stack

Primary: **Inter** (Google Fonts)
Monospace: **JetBrains Mono** (code samples, API keys)
